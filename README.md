# X-Man AI WordPress主题

一个现代化的WordPress主题，专为AI和技术博客设计。

## 目录结构

```
├── assets/          # 静态资源文件
│   ├── css/         # 样式文件
│   ├── images/      # 图片资源
│   └── js/          # JavaScript文件
├── docs/            # 文档目录
├── inc/             # 主题功能模块
├── template-parts/  # 模板片段
└── templates/       # 页面模板
```

## 主要功能

- 响应式设计
- 专题功能
- 软件展示页面
- 热门标签页面
- SEO优化
- 自定义小工具
